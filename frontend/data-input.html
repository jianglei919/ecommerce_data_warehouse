Order Items (order_items - Auto-generated)

Note: Order items are automatically generated based on order data
Mapping from order table to order_items table:
- product_id → order_items.product_id
- quantity → order_items.quantity
- total_amount/quantity → order_items.unit_price
- total_amount → order_items.subtotal

No need to manually input order_items data

📌 Instructions

  • ✓ Create Product: Requires product_name, category; brand is optional; no product_id
  • ✓ Update Product: Must include product_id, other fields optional (only include fields to update)
  • ✓ Create Order: Must include user_id, product_id, quantity, total_amount
  • ✓ Update Order: Must include order_id, other fields optional (only include fields to update)
  • ✓ Smart Partial Update: Remove unneeded fields, system only updates submitted fields and won't overwrite existing values
  • ✓ Auto-generated order_items: Created automatically when submitting an order
  • ✓ Batch Import: Send as an array [...]
  • ✓ Auto-sync to warehouse DB (dim_products, dim_orders)
  • ✓ Kafka Messages sent automatically (product-events, order-events)
