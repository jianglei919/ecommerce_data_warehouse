📋 数据模板

产品 (新增)

{
  "product_name": "iPhone 15 Pro",
  "category": "Electronics",
  "brand": "Apple"
}

产品 (编辑)

{
  "product_id": 1,
  "product_name": "iPhone 15 Pro"
}
// 仅更新需要更改的字段
// 不需要的字段可以删除，不会被更新

订单 (新增)

{
  "user_id": 1,
  "product_id": 1,
  "quantity": 1,
  "total_amount": 999.99,
  "status": "pending"
}

订单 (编辑)

{
  "order_id": 1001,
  "status": "shipped"
}
// 仅更新需要更改的字段
// 不需要的字段可以删除，不会被更新

订单项 (order_items - 自动生成)

注：订单项会根据订单数据自动生成
订单表中的字段映射到order_items表：
- product_id → order_items.product_id
- quantity → order_items.quantity
- total_amount/quantity → order_items.unit_price
- total_amount → order_items.subtotal

无需手动输入order_items数据

📌 说明

  • ✓ 新增产品：需要product_name、category；brand可选；不需要product_id
  • ✓ 编辑产品：必须包含product_id，其他字段可选（仅填写想要修改的字段）
  • ✓ 新增订单：必须包含user_id、product_id、quantity、total_amount
  • ✓ 编辑订单：必须包含order_id，其他字段可选（仅填写想要修改的字段）
  • ✓ 智能部分更新：编辑时不需要的字段可直接删除，系统只更新提交的字段，绝不覆盖已有值
  • ✓ order_items自动生成：提交订单时自动创建
  • ✓ 批量导入：发送数组形式 [...]
  • ✓ 自动同步到warehouse库（dim_products、dim_orders）
  • ✓ Kafka消息自动发送（product-events、order-events）
